# XML
Every class has:
1. Name – title of the class
2. Description – provides an overview of the class
3. Instructor – the instructor will have a name, email and phone
4. 4 Assignments – Each assignment will have a name and points
5. 2 Exams – Each exam will have a name and points
6. Grade – the final grade for the class. The grade can only take values – A or B or C
